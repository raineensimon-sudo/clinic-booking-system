# 🏥 Clinic Booking System

## 📘 Description
A simple MySQL database project that manages:
- Doctors
- Patients
- Appointments
- Medical Records
- Departments

## 🚀 How to Run

1. Install MySQL (or use XAMPP, MySQL Workbench).
2. Create a new database and run the SQL script:

```sql

USE clinic_db;
